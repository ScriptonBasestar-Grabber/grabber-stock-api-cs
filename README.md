Grabber API CS
==============

## 개요

증권 API 이용한 데이터 수집 및 저장

### 수집

* xing(ebestsec,이베스트투자증권)
* cybos(daishin,대신증권)

### 저장

* csv


## 환경구성

### DEV

사전설치 항목

* WIN10
* VS2019 - WPF, .NET desktop 개발도구
* XING API - 기본경로 설치
* XING DevCenter - RES 다운로드

소스코드 받은 후

1. git clone ~
2. GrabberCS.sln
3. nuget 초기화
4. DataLib이 안 열리는 경우 프로젝트 리로드
5. 노가다시작
